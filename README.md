# Stock Chaussures - version prête Vercel

Application React/Vite compatible Vercel.

## Installation locale
```bash
npm install
npm run dev
```

## Déploiement Vercel
1. Uploade tout ce dossier sur GitHub.
2. Sur Vercel, importe le repo.
3. Framework preset : Vite.
4. Clique sur Deploy.

Les données sont sauvegardées dans le navigateur avec localStorage.
